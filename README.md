# hermes-task-framework

Pip-installable wrapper for [task-framework](https://github.com/nuffin/task-framework) — structured task management skills for Hermes Agent.

## Install

```bash
pip install hermes-task-framework
```

## Skills

| Skill | Description |
|-------|-------------|
| task-framework | Core: methodology, container, tooling |
| task-tracker | Bookkeeping: TASK.md + TASK_MEMORY.md |
| task-timestamp-convention | Naming conventions |
| task-lifecycle-edge-cases | Recovery + resurrection |
| task-lifecycle-portability | Export/import/migration |
| task-external-repos-pattern | External repo cloning |
